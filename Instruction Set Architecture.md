- ## Definition
	- All of the **programmer visible** components and operations of the computer. For example:
		- [[Memory]] Organization
			- Address Space or number of addresses
			- Addressability or [[Word|Word Length]]
		- [[Register]] Set
			- Number of [[Register|Registers]]
			- [[Register]] size
		- [[LC3 Instruction]] Set
			- Opcodes 
			- Data types
			- Addressing models
