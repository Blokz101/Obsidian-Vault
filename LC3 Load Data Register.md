- ## Definition
	- Loads the value from a [[Memory Address]] to a [[Register]].
	  > [!Code]
	  > ```
	  > LDR DR SR1 OFFSET6
	  > ```
	  > 
	  >  Load the value from the memory location found by adding the value of `SR1` to `OFFSET6` into `DR`.
