#Permanent [[Calculus]]

- ## Definition
	- States that when a [[Limit]] of a function is an [[Indeterminate Type]] then the following rule applies.
	- > [!Equation] 
	  > $$\large{^{lim}_{x\rightarrow c}\frac{f(x)}{g(x)}=\ ^{lim}_{x\rightarrow c}\frac{f'(x)}{g'(x)}}$$
