#Permanent [[Chemistry]]

- ## Definition
	- [[Solid]]
		- Fixed shape
		- Fixed volume
		- Fixed [[Molecule|Molecules]]
	- [[Liquid]]
		- Variable shape
		- Fixed volume
		- Some [[Molecule|Molecular]] movement.
	- [[Gas]]
		- Variable shape
		- Variable volume
		- Fast [[Molecule|Molecular]] movement.
- ## Reference
	- Changes between these [[States Of Matter]] is called a [[Phase Change]].
