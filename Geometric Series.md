- ## Definition
	- An [[Infinite Series]] that takes the form of:
	- > [!Equation]
	  > $$\large{\sum^{\infty}_{n=0}ar^{n}}$$
	  > We need $n$ to start at $1$ to compute the sum so we can shift it like this:
	  > $$\large{\sum^{\infty}_{n=1}ar^{n-1}}$$
