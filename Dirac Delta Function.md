- ## Definition
	- > [!Equation]
	  > $$\large{\delta(t)=\lim_{ \tau \to 0 } \,p_{\tau}(t)}$$
	  > 
	  > $\delta(t)$ = [[Dirac Delta Function]]
	  > $p_{\tau}(t)$ = [[Unit Pulse Function]]
	  > $\tau$ = Time that [[Unit Pulse Function]] drops to $0$ from $1$
	- This function is character:
		- Zero Width
		- Infinite Hight
		- Area of $1$
