#Permanent [[Chemistry]]

- ## Definition
	- [[Humidity]] relative to the max [[Humidity]]. Can be calculated using [[Vapor Pressure]].
	> [!Equation]
	> $$\large{Relative\ Humidity=\frac{Observed\ Pressure_{H_20}}{Max\ Pressure_{H_20}}}$$
- ## Reference
	- Need [[Vapor Pressure]] to calculate.
