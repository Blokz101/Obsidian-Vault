- ## Definition
	- If $n$ pigeons are placed in $k$ boxes then there is at least one box that has $\left\lceil  \frac{n}{k}  \right\rceil$ pigeons in it.
		- See [[Ceiling Function]] if $\lceil x \rceil$ is unfamiliar.
